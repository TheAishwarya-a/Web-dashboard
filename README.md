## 📌 Project Overview

A fully interactive business intelligence dashboard simulating e-commerce analytics for 2023. Built with vanilla HTML, CSS, and Chart.js — no frameworks, no build step. Just open `dashboard.html` in a browser.

**Skills demonstrated:** Data visualization · Dashboard design · KPI storytelling · Chart.js · Responsive layout

---

## 📊 Dashboard Components

| Component | Chart Type | Insight |
|-----------|-----------|---------|
| 4 KPI Cards | Stat cards | Revenue, Orders, AOV, Margin |
| Monthly Trend | Bar + Line combo | Revenue & orders over 12 months |
| Category Split | Doughnut | Revenue share by category |
| Segment Bars | Bar chart | Consumer vs Corporate vs Home Office |
| Regional Radar | Radar chart | Multi-metric regional comparison |
| Payment Methods | Pie chart | UPI vs Cash vs Card vs Insurance |
